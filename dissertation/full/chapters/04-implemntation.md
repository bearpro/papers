# Реализация алгоритма

_В третей главе_ описана программная реализация алгоритмов. Реализация выполнена в виде двух модулей. Первый модуль отвечает за обработку XML-документов: реализован рекурсивный алгоритм обхода дерева, удаление пространств имён и построение строкового представления каждого элемента, что позволяет корректно сериализовать исходные данные. Второй модуль осуществляет получение эмбеддингов через алгоритмы энкодинга трансформеров, сохранение результатов в CSV, объединение данных из различных источников и последующую их обработку. Для визуализации результатов применяется интерактивное 3D-представление (Plotly), а метод EMD используется для оценки сходства эмбеддингов между группами. Такая интеграция программных и математических компонентов позволяет реализовать систему верификации целостности нормативных документов в производственной среде.
